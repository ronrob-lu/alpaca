# Alpaca Mod

A Minetest/Luanti mod that adds alpacas to the game!

## Features

- Alpacas spawn in the world.
- 4 different types of alpacas: White, Light Fawn, Dark Brown, and True Black.
- Each type has distinct genetics including speed, grass detection radius, drops, and energy drain.
- Reproduction and mutation mechanics.
- Alpacas drop wool when killed.

## How to use

You can spawn an alpaca using the spawn egg:
- Search for "Alpaca Spawn Egg" in creative mode.
- Or use the command `/giveme alpaca:spawn_egg` and then right click with the egg in your hand to spawn an alpaca.

## Dependencies

- default
- wool
